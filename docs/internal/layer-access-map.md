# Layer Access Map

## Layer 1 — Public

**Audience:** Anyone
**Content:** High-level docs, summaries, conceptual overviews

## Layer 2 — Internal (NDA)

**Audience:** Trusted collaborators
**Content:** Architecture, timing, arbitration, safety envelope

## Layer 3 — Operator-Only

**Audience:** Operator (Jason)
**Content:** Notes, recon logs, identity-linked material

## Layer 4 — Never Disclosed

**Audience:** Operator only
**Content:** Anything tied to personal lineage or identity
