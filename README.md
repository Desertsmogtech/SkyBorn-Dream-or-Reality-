# SkyBorn — Dream or Reality?

An open-source environmental monitoring and atmospheric research project.

## Project Structure

```
SkyBorn-Dream-or-Reality-/
├── mission.json          # Project mission and objectives
├── operator_script.txt   # Standard operating procedures
├── system_config.json    # System configuration and sensor settings
├── assets/               # Sensor data, logs, and raw assets
├── docs/                 # Documentation and changelogs
└── README.md             # This file
```

## Getting Started

1. Clone the repository
2. Review `mission.json` for project objectives
3. Follow `operator_script.txt` for operational procedures
4. Configure `system_config.json` for your environment
